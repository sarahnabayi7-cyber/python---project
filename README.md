# Python Games 🎮

## 📌 Description
This project contains a collection of simple command-line games built using python.
It was created to practice core programming concepts and improve problem-solving skills.
- Number Guessing Game
- Rock, Paper, Scissors
- True/False Quiz

## 🚀 How to Run
1. Make sure you have Python installed
2. Run a game:
```bash
python3 game.py
```
## 📂 Project Structure
- game.py → main game file
- python_games.py → collection of games
- qr.py → QR code generator

## 🎯 Features
- Simple CLI-based interaction
- Beginner-friendly code
- Multiple mini games
